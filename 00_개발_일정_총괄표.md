W1 종료 기준으로 아래 5개가 모두 완료되어야 W2 핵심 기능 구현으로 넘어간다.

| # | 항목 | 완료 기준 |
| --- | --- | --- |
| 1 | 저장소 운영 기준 | `dev` 브랜치, PR 템플릿, squash merge, 기본 보호 기준 준비 |
| 2 | 백엔드 골격 | `qtai-server` 단일 애플리케이션과 `common`, `config`, `security`, `batch`, `external`, `domain.member`, `domain.bible`, `domain.qt`, `domain.study`, `domain.note`, `domain.sharing`, `domain.report`, `domain.notification`, `domain.praise`, `domain.mission`, `domain.ai`, `domain.admin`, `domain.audit` 패키지 기준 생성 |
| 3 | 도메인 경계 검증 | 도메인 간 Entity/Service/Repository 직접 import 차단 테스트 준비 |
| 4 | DB/API 계약 | ERD 주요 테이블과 `/api/v1/**` 외부 API 초안 반영, 내부 Java Interface 분리 |

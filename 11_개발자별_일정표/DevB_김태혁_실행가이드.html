Dev B 역할 핵심 선언
"이 프로젝트의 데이터 심장은 성경 본문이다."
Bible Service는 Flutter 입체 묵상 화면과 AI 코칭 RAG의 기준 데이터다. 한/영 병기 조회, 주석 조회, Redis 24h 캐시를 안정화해 P95 ≤ 300ms를 지킨다.
담당핵심 책임
Passage책/장/절 좌표 기반 한글·KJV 병기 조회
CommentaryMatthew Henry 영문 주석 + 한글 더미 주석 조회
CacheRedis-Cache cache:passage:{book}:{chapter}:{verse} TTL 24h
SearchMySQL FULLTEXT 또는 JPQL LIKE 기반 구절 검색
🚫 저작권 가드레일
개역개정, ESV, NIV 사용 금지. 한국어 본문은 개역한글(비상업·교육·출처 표기) 또는 더미 데이터만 사용한다.
✅ 완료 증거
curl 응답, Flyway 로그, Redis TTL, 캐시 hit 로그, 검색 결과, Jacoco 리포트를 남긴다.

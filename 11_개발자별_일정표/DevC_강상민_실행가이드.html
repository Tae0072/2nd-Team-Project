"AI 없이는 이 앱이 존재하지 않는다."

SSEAsyncAnthropic 스트림과 첫 토큰 지연 관리
Prompt큐티 A~D 프롬프트와 신학 가드레일
Kafkaai.session.completed 발행과 schema envelope 준수
🚫 AI Service 혼용 금지
ai-service는 FastAPI 단독이다. Spring Boot WebClient/Java 서비스 코드를 만들지 않는다.

복붙 요청 프롬프트
[역할] QT-AI Dev C, FastAPI AI/RAG Service Owner. [요청] FastAPI 라우터, SessionUseCase, ChatUseCase SSE 골격을 작성해줘. 조건: - Python FastAPI 단독 - Java/Spring 코드 금지 - Pydantic 모델은 camelCase alias - ANTHROPIC_API_KEY는 env에서만 읽기 - 응답은 openapi.yaml과 일치
검증 체크
  • Spring Boot 코드가 섞이지 않았는지
  • API Key 하드코딩이 없는지
  • SSE가 실제 chunk로 흐르는지
  • ProblemDetail 형식과 충돌 없는지

  • 가드레일이 모든 단계에 포함됐는지
  • Bible API 경계를 지키는지

Flutter 역할 핵심 선언
"사용자가 처음 보는 것이 Flutter 화면이다."
시연에서 심사위원이 직접 손으로 만지는 유일한 인터페이스다. Sliver 기반 Sync Scroll과 AI SSE 스트리밍 실시간 타이핑 효과가 포트폴리오의 WOW 포인트가 된다. 백엔드 API가 완성되기 전에도 Mock 데이터로 화면을 미리 완성한다.
담당핵심 책임
NetworkDIO + JWT interceptor + refresh flow
BibleSliver Sync Scroll 한/영 병기 화면
AI ChatSSE 수신 + 타이핑 애니메이션
Journal/Home묵상 노트 CRUD와 대시보드 흐름
🚫 화면 선행 원칙
백엔드가 늦어져도 Mock 데이터로 UI와 상태 흐름을 먼저 완성한다. API 응답 모델은 OpenAPI camelCase와 맞춘 뒤 교체한다.

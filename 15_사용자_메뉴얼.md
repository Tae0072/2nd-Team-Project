## 1. 시작하기

| 상황 | 사용 방법 |
| --- | --- |
| 처음 실행 | 로그인 화면으로 이동하고, Kakao 로그인 후 튜토리얼을 거쳐 Today QT로 이동한다. |
| 미인증 | 앱 기능 화면에 접근할 수 없고 로그인 화면으로 이동한다. |
| 로그인 후 | Today QT, 성경 본문, 해설 보기, 묵상 저장, 찬양 저장, 묵상 달력 조회를 사용할 수 있다. |

---

## 2. Today QT 보기

| 기능 | 안내 |
| --- | --- |
| 오늘 본문 | 앱 첫 화면에서 오늘 QT 본문을 확인한다. |
| 영어 본문 | 필요할 때 펼쳐서 한글 본문과 함께 본다. |
| 해설 보기 | 로그인 후 쉬운 해설과 출처를 확인한다. |
| 묵상 작성 | 오늘 QT 기준으로 느낀 점, 기억할 구절, 적용, 기도를 작성한다. |
| 시뮬레이터 보기 | 버튼이 활성화된 경우 사전 생성 클립을 본다. |

시뮬레이터 버튼이 비활성화되어 있어도 Today QT가 실패한 것은 아니다. 상태값은 정상 제공되며, 클립이 준비되지 않았거나 제외된 본문일 수 있다.

---

## 3. 묵상 기록

| 기능 | 안내 |
| --- | --- |
| 작성 | 4개 항목 중 하나 이상 입력하면 저장할 수 있다. |
| 수정 | 묵상 상세 화면에서 내용을 수정한다. |
| 삭제 | 본인 기록만 삭제할 수 있다. |
| 달력 | 마이페이지에서 묵상한 날짜를 확인한다. |
| 닉네임 나눔 | 사용자가 선택한 경우에만 닉네임이 노출된 공유본을 게시한다. |

---

## 4. 찬양 큐레이션

| 기능 | 안내 |
| --- | --- |
| 조회 | 운영자가 미리 등록한 찬양 목록을 본다. |
| 저장 | 로그인 후 내 찬양 목록에 저장한다. |
| 제한 | AI 추천, 가사 저장, 음원 저장, 직접 URL 입력은 제공하지 않는다. |

---

## 5. 제공하지 않는 기능

| 기능 | 안내 |
| --- | --- |
| AI 자유 챗봇 | 다중 턴 상담형 챗봇과 SSE 스트리밍은 제공하지 않는다. |
| 교회 인증 | MVP에는 교회 인증 기능이 없다. |
| 자유 본문 묵상 | 묵상 작성은 오늘 QT 본문 기준이다. |
| 팔로우·랭킹 | 개인 묵상 중심으로 운영한다. |

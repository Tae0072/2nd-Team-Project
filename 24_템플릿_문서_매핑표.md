## 1. 매핑 원칙

| 원칙 | 내용 |
| --- | --- |
| 요구사항 원본 우선 | 기능·비기능·화면 요구사항은 `07_요구사항_정의서.md` v2.3을 우선한다. |
| 템플릿은 형식 참고 | 외부 템플릿은 산출물 종류와 목차 참고용이며, QT-AI의 결정과 충돌하면 QT-AI 문서를 우선한다. |
| 중복 최소화 | 같은 내용을 여러 문서에 복사하지 않고, 각 문서의 책임만 남긴다. |
| 번호 충돌 회피 | 템플릿 번호를 우선해 시퀀스는 `05_시퀀스_다이어그램.md`, 요구사항은 `07_요구사항_정의서.md`로 분리한다. |
| 보강 문서 분리 | 템플릿에 없는 기능 명세, 용어사전, 구현 저장소 체크리스트는 별도 번호로 둔다. |

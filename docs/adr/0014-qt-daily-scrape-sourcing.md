# ADR-0014: QT 본문 일일 스크래핑 수집 (성서 유니온, 19:00)

## 상태
Accepted (2026-05-14)

## 날짜
2026-05-14

## 작성자
강태오

## Reviewer
이지윤(조사 담당), 김태혁, 강상민, 이승욱, 김지민

## Context
2025-01-01 성서 유니온이 자체 QT 서비스를 종료했고, 두란노 등 한국 대표 QT 출판사의 B2B 콘텐츠 공급은 연 수백만~수천만 원 규모로 학생 팀이 부담할 수 없다. 동시에 기존 무료 QT 앱들은 다수가 본문 전문을 무단 스크래핑하는 방식으로 운영되어 왔다. 2026-05-13 오후 회의에서 강사는 "본문 전문이 아닌 그날치 좌표만 가져오는 방식이라면 위험을 최소화할 수 있다"는 방향을 제시했다. 2026-05-14 회의에서 이지윤 조사 결과로 출처·갱신 시간·범위가 확정되었다.

**합법성 단정 금지.** 본 ADR은 "합법"이라고 단정하지 않는다. robots.txt·이용약관 확인 전까지는 좌표 메타데이터만 최소 수집하고, 출판사 측 명시적 거부가 있으면 즉시 중단한다.

## Decision

| 항목 | 결정 |
| --- | --- |
| **출처** | **성서 유니온** (수영로 교회 등 사용 출판본과 동일 본문 확인) |
| **가져오는 방식** | **최소 메타데이터 수집** (좌표만, 본문 텍스트는 자체 DB) |
| **가져오는 범위** | 그날 본문 좌표만 (예: "창세기 41:37–57"). 본문 텍스트는 가져오지 않음 |
| **본문 데이터** | 자체 DB `qtai_db`의 `bible_kr_verses` / `bible_en_verses`에서 셀렉트 |
| **갱신 시간** | **저녁 19:00** (다음 날 본문 공개) |
| **수집 cron** | `0 19 * * *` |
| **결과 적재 위치** | `bible_today_qt_schedule` 테이블 (`qt_date`, `book_code`, `chapter`, `verse_start`, `verse_end`) |
| **법적 정책** | robots.txt·이용약관 확인 전까지 좌표 메타데이터만 최소 수집. 본문 텍스트는 자체 DB에서 조회. 출판사가 명시적 거부하면 즉시 중단 |
| **실패 처리** | 19:00 수집 실패 시 30분 간격으로 3회 재시도. 모두 실패하면 운영자 알림 (관리자 페이지) + 다음 날 06:00 폴백 시도 |

## Alternatives
- **두란노 B2B 라이센스 구매**: 비용 부담 과도. 학생 프로젝트 범위 외.
- **AI 생성 QT 본문**: 신학 검증·저작권 리스크 모두 큼. 묵상의 가치 손상.
- **사용자가 직접 본문 입력**: MVP 첫 화면을 오늘 QT로 결정한 5/13 결정과 정합 안 맞음.
- **다른 무료 QT 앱 데이터 재사용**: 그들 자체가 스크래핑 기반 — 출처 신뢰성 떨어지고 합법성 보장 안 됨.

## Consequences
**긍정:**
- MVP 첫 화면(오늘 QT)의 데이터 공급 확보.
- 본문 텍스트는 자체 DB(개역한글/KJV)에서 셀렉트하므로 저작권 안전.
- 스크래핑 대상이 좌표(메타)뿐이라 페이로드 작고 안정적.

**부정:**
- 성서 유니온 페이지 구조 변경 시 스크래퍼 깨질 위험 → DOM 셀렉터 변경 감지 알림 필요.
- 새번역 등 추가 번역본 채택 시 본문 DB 적재 별도 작업 (ADR 별도 또는 DECISIONS §8 갱신).
- 합법성 근거는 "무료 공개 범위 내 수집"이라는 일반적 해석이라 출판사가 명시적 거부 시 재검토 필요.

## 검증 방법
W2 5/29까지:
1. `bible_today_qt_schedule` 테이블에 매일 19:00 그날 좌표 1행 추가됨.
2. 19:00 ± 30분 윈도우 안에 cron 잡이 동작 (로그·메트릭으로 확인).
3. 페이지 구조 변경 시뮬레이션 테스트(셀렉터 깨짐) → 알림 발생.

## 후속 결정 필요
- **새번역 라이선스 명시적 확인** — 대한성서공회와 라이선스 협의 후 사용 여부 결정 (이지윤).
- **영어 본문 적재 방식** — 무료 API 직접 호출 vs PDF 다운로드 → 둘 다 시도 후 채택 (이지윤 추가 조사).
- **스크래핑 robots.txt·이용약관 확인** — 성서 유니온 페이지의 명시적 이용 정책 확인.
